# dapp-base
a base project of decentralized apps


# How to use

1. `docker-compose build`
2. `docker-compose run truffle npm install`
3. `docker-compose run truffle truffle migrate`
4. `docker-compose up`
5. open `http://localhost:8003/` in your browser

# Dependencies

- Node.js
- Web3.js
- React.js
- Truffle
- Ganache
